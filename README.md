🏡 Airbnb Clone – Backend (Spring Boot)

A scalable and secure backend system for an Airbnb-like accommodation booking platform built using Spring Boot.
This project handles user authentication, property listings, booking management, and payment integration with a focus on clean architecture and RESTful APIs.
🚀 Features
🔐 User Authentication & Authorization (Spring Security + JWT)
🏠 Property Listing Management (Add, Update, Delete listings)
📅 Booking System (Availability check & reservations)
💳 Payment Integration (Stripe/Razorpay ready)
⭐ Reviews & Ratings System
📍 Location-based Search
📊 Admin Dashboard APIs
📩 Email Notifications (Booking Confirmation, Alerts)
🛠️ Tech Stack
Backend: Spring Boot, Spring MVC
Security: Spring Security, JWT
Database: MySQL / PostgreSQL
ORM: Hibernate (JPA)
Build Tool: Maven
API Testing: Postman
📂 Architecture
Layered Architecture (Controller → Service → Repository)
RESTful API Design
DTO-based request/response handling
Exception handling & validation
⚡ Key Highlights
Clean and maintainable code structure
Scalable backend design for real-world applications
Secure API endpoints with role-based access
Optimized database queries using JPA
📌 Use Case

This backend powers a full-stack Airbnb clone where users can:

Browse properties
Book stays
Manage listings (for hosts)
Leave reviews
